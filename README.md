## Getting Started

### Prerequisites

* Windows 10 / Windows 11
* Pixel Gun 3D (Steam version)
---

### Setup (Manual Injection)

1. Download `Moonlite.dll`
2. Open your preferred injector
3. Launch Pixel Gun 3D
4. Inject into:

```

```
